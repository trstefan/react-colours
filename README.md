## 📝 Project Description

Colours generator built in React using Hooks. 

## 📚 Technologies

  - React
  - React Hooks (useState) 
  - CSS
  
## 🎯 Features
  - Generate a random colour
  - Copy colour's code 

## 🏃 Run locally

Clone the project

```bash
  git clone https://github.com/trstefan/react-colours.git
```

Go to the project directory

```bash
  cd react-colours
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
